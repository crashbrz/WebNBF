# WebNBF #
Simple Bash "Multitask" Web NTLM Authentication Bruteforcer.

### Usage/Help ###

```
./webnbf.sh <userlist> <passwordlist> https://target

```

### Proxy ###
Add or remove the curl proxy parameters according to your environment.
